2017年9月28日

你以为这是一篇性能调优博客？No 其实这只是一份吐槽~

问题描述：公司网站（运营系统）进入后一段时间访问速度奇慢，刚刷新进入时并不会出现此现象

针对此问题做了很多调试工作，最后原因是php 写入数据库时，插入数据失败，此时应该是无法获取token而退出，由于链接写死的原因，导致无法弹出已到期的提示。
